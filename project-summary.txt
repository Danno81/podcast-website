=====================================================

==================

├── index.html          (8.5 KB)  - Home page
├── episodes.html       (8.2 KB)  - Full episode listing
├── about.html          (13 KB)   - About podcast & host

TECHNICAL IMPLEMENTATION:
=========================

RSS Feed Handling:
  - Source: https://rss.buzzsprout.com/2324798.rss
  - CORS Proxy: https://api.allorigins.win/raw?url=...
  - Client-side parsing with DOMParser
  - 1-hour caching to reduce requests
  - Graceful error handling

JavaScript Features:
  - Async/await for clean code
  - Error handling & fallbacks
  - Dynamic HTML creation
  - Event listeners for interactivity
  - Search/filter functionality
  - No external libraries or frameworks

Podcast Integration:
  - Buzzsprout hosting page
  - YouTube channel
  - Apple Podcasts link
  - Spotify link
  - Google Podcasts link
  - Direct audio player
  - Email contact
  - Speakpipe voicemail
  - LinkedIn & Bluesky social
